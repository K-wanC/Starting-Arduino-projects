The whole program is based on the button and INPUT_PULLUP mechanism

-INPUT_PULLUP: A existing mechanism within arduino board, can be used via code; it connects the pin assigned "INPUT_PULLUP" to a power with resistor, usually relatively high, so when one side of the current is connected to GND, the input read will be 0 due to high resistance from the resistor.

-button: Connects two side while pushing, connects only vertical while not pushing.
